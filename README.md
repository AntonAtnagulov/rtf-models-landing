### rtf-models-landing

---
![alt](./Landing.gif)

An example of an advertising page with a preview of a 3D model.
React-three-fiber was used to render 3d models.

---

###### Technologies

- React 18.1.0
- React-three-drei 9.11.2
- React-three-fiber 8.0.18

---

For start:
```
cd client
npm install
npm start
```
